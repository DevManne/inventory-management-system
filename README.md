Inventory Management System

A responsive Inventory Management System built with React, useReducer, Recharts, and Vite. Supports PWA offline mode.

##  Features
- Search & filter products by name, category, stock level
-  Low-stock dashboard with critical/warning alerts
- Stock charts by category and product
- Sell and restock products
- Export inventory as JSON
-  PWA support (offline ready)

##  Tech Stack
- React 18
- useReducer for state management
- Recharts for data visualization
- Vite for bundling
- Service Worker for PWA

##  Project Structure
src/        → React components & logic
public/     → Static assets & PWA files
## Run Locally
```bash
npm install
npm run dev
```
